# Pourō-GPT Re

Repository bootstrap foundation for the Pourō-GPT Re client application.

## Architecture Baseline

- Client-side PWA-first single-page application.
- React, TypeScript strict mode, and Vite.
- pnpm through Corepack on Node.js 24.
- Plain CSS.
- Vitest, Testing Library, and Playwright Chromium for bootstrap validation.

Product UI, routing, state management, PWA manifest/service worker, assets, GitHub Actions, Recipe Truth implementation, and Active Brew behavior are outside PR-A scope.

## Commands

```text
pnpm install
pnpm format:check
pnpm lint
pnpm typecheck
pnpm test
pnpm build
pnpm test:e2e
pnpm ci
pnpm validate
```
